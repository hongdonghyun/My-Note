2. 데이터베이스에서 데이터를 추출하는데 사용되는 SQL문은 무엇인가?

3. 데이터베이스의 데이터를 업데이트하는 데 사용되는 SQL문은 무엇인가?

4. 데이터베이스에서 데이터를 삭제하는 데 사용되는 SQL문은 무엇인가?

- DELETE

5. 데이터베이스에 새 데이터를 삽입하는데 사용되는 SQL문은 무엇인가?
