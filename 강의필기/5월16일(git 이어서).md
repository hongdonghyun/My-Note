##저장소 나눠쓰기 (공유)

remote 저장소에서 내용을 가져오려면
```
git fetch origin
```
저장소에 있는 내용을 가지고만 오는것
```
git merge origin/master
```
실제 적용해주는 명령어 (합치기)

한번에 해주는 명령어는
```
git pull origin
```
